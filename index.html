關於我
大家好，我是來自台中二中的阮香綾。
我在選組時選擇了第二類組，因為對資訊領域充滿好奇與熱情。
在校內的資訊課中，第一次接觸到了程式設計，開始透過 ZeroJudge 練習題目，加深了對邏輯思考與程式語法的理解。
除此之外，我也利用課餘時間自學網頁設計，實作簡單的靜態網頁，
運用了 HTML、CSS 等基礎技術。這段學習經歷讓我發現寫程式的樂趣，
也培養了解決問題的能力。
目前我正在嘗試更進一步地接觸 JavaScript 與網頁互動設計，
希望未來能學習更多前端框架，甚至挑戰後端開發，
打造出完整的網頁作品。
我期待持續精進 coding 技能，也希望未來能參與實際專案，
累積更多實戰經驗，朝資訊領域邁進。
